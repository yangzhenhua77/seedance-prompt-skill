# Seedance 视频提示词 Skill — Claude Code 插件

一个 Claude Code 自定义 Skill，让 Claude 成为专业的 AI 视频提示词工程师，专为字节跳动 **Seedance 2.0**（即梦）视频生成平台打造。

只需用自然语言描述你的创意想法，Skill 会自动生成结构化、可直接复制到即梦平台使用的高质量中文视频提示词，助你产出电影级 AI 视频。

## 功能特性

### 十大核心提示词生成能力

| # | 能力 | 说明 |
|---|------|------|
| 1 | **纯文本生成** | 无需任何参考素材，纯靠文字描述生成视频 |
| 2 | **一致性控制** | 通过上传参考图片，保持角色/产品/场景在多镜头间的统一 |
| 3 | **运镜与动作复刻** | 上传参考视频，精准复刻其中的镜头语言、复杂动作和节奏变化 |
| 4 | **创意模板/特效复刻** | 照着参考视频模仿创意转场、广告成片、电影片段 |
| 5 | **剧情创作/补全** | 根据分镜图片或脚本自动生成完整剧情 |
| 6 | **视频延长** | 对已有视频进行前向或后向平滑延长 |
| 7 | **声音控制** | 音色参考、对白生成、音效设计 |
| 8 | **一镜到底** | 生成连贯长镜头，从一个场景平滑过渡到另一个场景 |
| 9 | **视频编辑** | 基于已有视频进行角色替换、剧情颠覆、元素增减 |
| 10 | **音乐卡点** | 让画面节奏与音乐节拍精准匹配 |

### 高级技巧

- **时间戳分镜法** — 按秒精确控制 15 秒视频的每个镜头（如 `0-3秒：...`、`4-8秒：...`）
- **技术参数指定** — 在提示词开头声明画幅比、帧率、色调、镜头风格
- **禁止项声明** — 明确排除不需要的元素（水印、字幕、Logo 等）
- **超长视频分段拼接** — 自动将超过 15 秒的视频拆分为多段，通过视频延长串接，并附带衔接点描述

### 多模态引用系统

支持 Seedance 2.0 的 `@` 引用语法，组合多种输入模态：

- **图片**（`@图片1` ~ `@图片9`）— 角色参考、场景背景、首帧/尾帧控制
- **视频**（`@视频1` ~ `@视频3`）— 运镜、动作、特效、节奏参考
- **音频**（`@音频1` ~ `@音频3`）— 背景音乐、音色、音效参考

### 场景化提示词策略

- **电商/广告** — 产品 360 度旋转展示、爆炸分解、3D 渲染特效
- **短剧/对白** — 剧本化场景，含角色台词、情绪标注和音效设计
- **仙侠/奇幻动画** — 电影级武打、法术特效、变身变装序列
- **科普教学** — 4K 医学 CGI、半透明人体结构可视化
- **MV/音乐卡点** — 宽银幕构图、多图卡点参考视频节奏

### 内置提示词素材库

Skill 内置了专业的词汇库：

- **镜头语言** — 景别、运镜、角度、节奏、焦点、转场
- **视觉风格** — 画面质感、影像风格、色调氛围、艺术风格、光影效果、动画风格

## 安装

### 前置要求

- 已安装并登录 [Claude Code CLI](https://docs.anthropic.com/en/docs/claude-code)
- 拥有 [即梦平台](https://jimeng.jianying.com) (Seedance 2.0) 账号，用于使用生成的提示词

### 安装步骤

1. **克隆仓库**

```bash
git clone https://github.com/songguoxs/seedance-prompt-skill.git
```

2. **将 Skill 文件复制到你项目的 `.claude/skills/` 目录**

```bash
# 进入你要使用该 Skill 的项目目录
cd /path/to/your/project

# 创建 skills 目录（如不存在）
mkdir -p .claude/skills/seedance

# 复制 Skill 文件
cp /path/to/seedance-prompt-skill/.claude/skills/seedance/SKILL.md .claude/skills/seedance/SKILL.md
```

3. **配置 Claude Code 权限**

在项目根目录创建或编辑 `.claude/settings.local.json`：

```json
{
  "permissions": {
    "allow": [
      "WebSearch"
    ]
  }
}
```

> `WebSearch` 权限为可选但推荐开启 — 允许 Skill 在需要时查询即梦平台的最新动态。

4. **验证安装**

在项目目录启动 Claude Code：

```bash
claude
```

输入 `/seedance` 或直接描述你的视频创意。Claude 会自动激活 Seedance 提示词 Skill。

### 全局安装（可选）

如果希望在所有项目中都能使用该 Skill，将其放置在用户主目录下：

```bash
mkdir -p ~/.claude/skills/seedance
cp /path/to/seedance-prompt-skill/.claude/skills/seedance/SKILL.md ~/.claude/skills/seedance/SKILL.md
```

## 使用方法

### 快速开始

启动 Claude Code，描述你想要创建的视频：

```
You: 帮我生成一段赛博朋克风格的城市夜景视频提示词
```

Claude 会先询问几个关键参数（时长、画面比例、参考素材情况），然后生成 2-3 个版本的提示词，可直接复制到即梦平台使用。

### 触发方式

Skill 会在以下情况自动激活：

- 使用 `/seedance` 命令
- 提到关键词：`Seedance`、`即梦`、`视频提示词`、`视频生成`、`AI视频`、`短剧`、`广告视频`、`视频延长`
- 要求"生成视频提示词"或"写视频描述"

### 交互流程

Skill 遵循结构化的四步工作流：

```
第一步：描述创意 → "一段古风武侠打斗"
第二步：确认参数 → Claude 询问时长、比例、参考素材、风格偏好
第三步：生成提示词 → ≤15秒生成 2-3 个版本，>15秒输出多段拼接方案
第四步：微调优化 → 调整时间段画面、镜头语言、台词、音效或风格
```

## Seedance 2.0 平台规格

| 维度 | 规格 |
|------|------|
| 图片输入 | jpeg/png/webp/bmp/tiff/gif，最多 9 张，单张 <30MB |
| 视频输入 | mp4/mov，最多 3 个，总时长 2-15 秒，单个 <50MB，分辨率 480p-720p |
| 音频输入 | mp3/wav，最多 3 个，总时长 ≤15 秒，单个 <15MB |
| 文本输入 | 自然语言描述 |
| 文件上限 | 最多 12 个文件（图片 + 视频 + 音频合计） |
| 生成时长 | 4-15 秒，可自由选择 |
| 声音输出 | 自带音效/配乐 |
| 分辨率 | 支持 2K 输出 |

> **重要提示：** 平台不支持上传含有写实真人脸部的素材（图片和视频均不可），系统会自动拦截。

## 最佳实践

- **描述要具体有画面感** — "穿红风衣的女人走过雨后霓虹闪烁的街道"远比"一个女人在走路"效果好
- **13-15 秒视频使用时间戳分镜** — 按秒划分镜头，精确控制每段画面内容
- **台词和音效单独描述** — 与画面描述分行书写，结构更清晰，生成效果更好
- **结尾添加禁止项** — 如"禁止出现水印、字幕、Logo"
- **参考图片风格要匹配主题** — 例如古风主题用水墨风格图片，赛博朋克主题用科幻 CG 图片
- **超过 15 秒的视频** — 交给 Skill 自动拆分为多段，每段之间会附带明确的画面衔接点描述
- **台词用引号包裹** — 并标注角色名称和情绪（如"总裁，卑微慌乱"）
- **情绪氛围不要忽略** — 情绪和氛围的描述对最终效果影响很大

## 常见问题

### Skill 没有自动激活怎么办？

确保 `SKILL.md` 文件放置在正确的路径下：`.claude/skills/seedance/SKILL.md`。如果仍然无法激活，尝试直接使用 `/seedance` 命令触发。

### 生成的提示词效果不理想？

- 尝试提供更具体的描述，避免过于抽象的表述
- 使用时间戳分镜法精确控制每段画面
- 在提示词结尾添加禁止项，排除不想要的元素
- 让 Claude 生成多个版本，挑选最合适的进行微调

### 支持英文提示词吗？

目前 Skill 专为中文提示词优化，因为 Seedance 2.0 平台对中文自然语言理解能力最强。生成的所有提示词（包括视频提示词和配套的图片生成提示词）均为中文。

### 超长视频如何保证分段之间的连贯性？

Skill 会在每段提示词后附带「衔接点」描述，精确描述该段结尾的画面状态。下一段的开头会自然承接上一段的结尾，通过 Seedance 2.0 的视频延长功能实现平滑过渡。
